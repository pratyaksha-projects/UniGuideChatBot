# UniguideChatBot

A campus navigation chatbot for HBTU Kanpur — find departments, faculty, facilities and more just by asking.

Built with LangChain + HuggingFace + Streamlit.

---

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

Get a free HuggingFace token from [huggingface.co/settings/tokens](https://huggingface.co/settings/tokens) and paste it in the sidebar.

> Campus data is in `data/hbtu_data.json` — edit it to add faculty or update locations.
